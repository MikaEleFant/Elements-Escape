Elements Escape
A text-based escape game based on the elements!

How To Play:

Choose an action followed by the corresponding item(s) to interact with:

  • USE: Choose an item to use on another item, prompted by the second input. Order is important here! For example, "USE key ON door".
  • INSPECT: Choose an item to closely inspect. Certain items require inspection before they can be used or combined, and can sometimes reveal new items. For example: "INSPECT key".
  • PICK UP: Choose an item to pick up and place in your inventory. Items you pick up will be displayed in the ITEMS section. Certain items need to be picked up in order to be used or combined. For example: "PICK UP key".
  • COMBINE: Choose two items to combine in order to create a new item. Although order is not important here, note that there is a difference between combining items and using items. For example: "COMBINE door WITH doorknob".

You must obtain the element required by the door in order to escape. Use your surroundings to your advantage, and exercise your elemental knowledge! Good luck!
