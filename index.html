Elements Escape

A text-based escape game based on the elements!

By Chang Ho Lee @MikaEleFant

Objective:

Each door has a number imprinted on it, and a scanner next to it.

This number corresponds to an element on the Periodic Table.

You must scan the element required by the door in order to escape.

If the element requires a container, the scanner will have a groove in the shape of the container required.

Use your surroundings to your advantage, and exercise your elemental knowledge!

Good luck!



How To Play:

Type in your next action in the input box. Make sure to format your input correctly as shown:

  • USE: Choose an item to use. For example: "USE lighter". If you wish to use an item on another item, follow it up with the "on" keyword. For example: "USE key ON door". Order is important here! Although you may only use an item in your inventory, you can use an inventory item on an object not within your inventory.
  • INSPECT: Choose an item to closely inspect. Certain items require inspection before they can be used or combined, and can sometimes reveal new items. For example: "INSPECT key".
  • PICK UP: Choose an item to pick up and place in your inventory. Items you pick up will be displayed in the ITEMS section. Items need to be picked up in order to be used or combined. For example: "PICK UP key".
  • COMBINE: Choose two items to combine in order to create a new item. Although order is not important here, note that there is a difference between combining items and using items. For example: "COMBINE door WITH doorknob". Remember, both items must be within your inventory in order to combine them!

That's it! Enjoy the game! HINT: You will need to PICK UP items after you USE or COMBINE them!
